# Scenario Design Phase 2: Sectors 7-13 (Complex Logic)

## Sector 7: Firewall Alpha (Optics)
*   **Goal**: Pass the RGB authentication.
*   **Steps**:
    1.  [Console] demands 3 keys: Red, Green, Blue.
    2.  Find **[Prism]** in trash.
    3.  Find **[White Laser]** (or light source).
    4.  Use [Prism] on [Laser] -> Splits light -> Get **[RGB Beams]**.
    5.  Wait, physical items needed.
    6.  Find **[Red Chip]**.
    7.  Find **[Blue Chip]**.
    8.  find **[Yellow Chip]**.
    9.  Mix? No.
    10. **Puzzle**: The slot for Green is broken. You must mix Blue + Yellow paint?
    11. **Final Logic**:
        *   Find **[Clear Chip]**.
        *   Find **[Blue Paint]**, **[Yellow Paint]**.
        *   Combine Paint -> **[Green Paint]**.
        *   Paint Chip -> **[Green Chip]**.
        *   Insert Red, Green, Blue Chips.

## Sector 8: Incinerator (Remote Control)
*   **Goal**: Press button on far side of fire pit.
*   **Steps**:
    1.  Bridge retracted.
    2.  Find **[Toy RC Car]** (Broken).
    3.  Find **[Controller]** (No signal).
    4.  Find **[Antenna]** (Wire). Use Wire on Controller -> Fix.
    5.  Car is too light to press button.
    6.  Find **[Lead Weight]**. Put on Car.
    7.  Use **[Controller]** -> Drive Car -> Press Button.

## Sector 9: Assembly Line (The Impostor)
*   **Goal**: Pass "Quality Assurance" scanner.
*   **Steps**:
    1.  Scanner rejects you ("Organic detected").
    2.  Find **[Robot Shell]** (Chest piece).
    3.  Find **[Data Cable]** (To connect).
    4.  Find **[Barcode 001]** (Verified user).
    5.  Attach Shell. Attach Barcode.
    6.  Step on scanner -> "Verified Machine".

## Sector 10: The Index (Silence)
*   **Goal**: Get the Key Book without making noise.
*   **Steps**:
    1.  Librarian kills on sound.
    2.  Floorboards creak at sector [A-3].
    3.  Find **[Lubricant]** (Grease).
    4.  Grease the floor -> Silent walk.
    5.  Book is high up. Ladder is squeaky? No.
    6.  Book is locked by "Dewey Decimal".
    7.  Find **[Catalog Card]** -> Read code "808.8".
    8.  Find Shelf 808 -> Get **[Key Book]**.

## Sector 11: Redacted Zone (Encryption)
*   **Goal**: Read the Blacked-out text on the wall.
*   **Steps**:
    1.  Text is covered in digital [Black Ink].
    2.  Find **[Scanner]** tool.
    3.  Scan text -> "Encrypted".
    4.  Find **[Decryption Module]** (Broken).
    5.  Find **[Spare Capacitor]**. Fix Module.
    6.  Combine Scanner + Module.
    7.  Scan again -> Text reveals password.

## Sector 12: Echo Room (Feedback)
*   **Goal**: Crash the audio loop system.
*   **Steps**:
    1.  Room repeats every sound.
    2.  Find **[Microphone]**.
    3.  Find **[Speaker]**.
    4.  Plug Mic into Speaker.
    5.  Turn on -> **Audio Feedback Loop** (Screech).
    6.  System crashes ("Stack Overflow"). Door opens.

## Sector 13: Glitch Storm (Debugging)
*   **Goal**: Freeze the storm.
*   **Steps**:
    1.  Storm acts like a buggy game object (flickering).
    2.  Find **[Debugger Tool]** (Gun like).
    3.  Aim at Glitch -> "Target Locked".
    4.  Fire -> "Breakpoint Set".
    5.  Glitch freezes in place.
    6.  Use **[Patch Cable]** to bridge the frozen glitch.
